Here's a README for WacOS:


A desktop OS built with HTML, CSS, and JavaScript - Inspired by windows and mac so its basically the diferent features combines. Its not that apparent right now but in WEBOS2 I will make it a lot more noticable 

Features

Draggable, closeable, and maximizable windows(mac inspired close)
timer app
stopwatch app
information 
Live clock in the corner
